# VIP Клуб Мусульман — Landing

Закрытое сообщество мусульман-предпринимателей. Одностраничный лендинг,
полностью self-contained: HTML + встроенные CSS/JS, без внешних зависимостей
(изображения встроены как data-URI).

## Структура

- `index.html` — весь сайт в одном файле.

## Локальный просмотр

Открыть `index.html` в браузере напрямую, либо поднять локальный сервер:

```bash
npx serve .
```

## Деплой

Сайт опубликован через **GitHub Pages** из ветки `main` (корень репозитория).
Любой `git push` в `main` автоматически обновляет прод.
