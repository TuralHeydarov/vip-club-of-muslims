# Unity Клуб Мусульман — Landing

Закрытое сообщество мусульман-предпринимателей. Одностраничный лендинг,
полностью self-contained: HTML + встроенные CSS/JS, без внешних зависимостей
(изображения встроены как data-URI).

## Структура

- `index.html` — весь сайт в одном файле.
- `netlify.toml` — настройки публикации (статика, без сборки).

## Локальный просмотр

Открыть `index.html` в браузере напрямую, либо поднять локальный сервер:

```bash
npx serve .
```

## Деплой

Сайт опубликован на **Netlify**.

**Live:** https://unity-muslims-club.netlify.app

Сейчас деплой выполняется вручную из чистой папки публикации:

```bash
npx netlify-cli deploy --prod --dir <папка-с-index.html>
```

> Авто-деплой из GitHub (push в `main` → автообновление) пока не подключён.
